- Express
- REST API
- Axios

## Features
- Make API request to an external API
- Stores history of names written by the user



## Installation

Install Name App Backend with npm

```bash
  git clone https://github.com/shaniKupiec/Name-App-Backend.git
  cd Name-App-Backend
  npm i
  npm start
```
